60 / 120 FPS 极速绘制

使用 Virtual Math Tree (VMT) 拓扑与渲染解耦架构。提供 Canvas2D 合并批处理与 WebGL/WebGPU 实例化管线，彻底打碎 Draw-Call 物理瓶颈。

Zero-DOM 零重排损耗

整个 UI 树运行在单个 Canvas 画布内，排版计算全在 JS/Worker 层执行，避开了浏览器漫长的 DOM 重排与重构（Reflow/Reflow Thrashing）。

首创双轨 A11y 影子映射

突破传统 Canvas 图表的无障碍盲区。通过动静隔离同步算法，在 Canvas 后台投影出透明的 A11y 影子 DOM，无缝支持屏幕阅读器和 Playwright 自动化测试。
